# AGENTS.md

## Scope
This repository contains only public, player-facing files for the Kaiki Sign Game.

## Hard rules
- Do not add spoilers, judge rules, endings, or GM secret prompts to this repository.
- Do not add unpublished episode materials.
- Keep each episode under `docs/episodes/epXX/`.
- Keep the public site static: HTML, CSS, JavaScript, JSON, and images only.
- Do not add build tooling unless explicitly requested.

## Episode rules
- Public episode data must be stored in `episode.json`.
- Image files shown to players should use neutral labels such as `A`, `B`, `C`.
- Do not expose exact pictogram meanings unless the episode intentionally displays them.
- The public page should guide viewing, not judge answers.

## Reporting
When modifying this repo, update `REPORT.md` with:
- changed files
- confirmed checks
- unconfirmed items
- follow-up items
