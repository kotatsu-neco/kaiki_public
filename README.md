# kaiki-sign-game public starter

GitHub Pages 用の公開リポジトリ雛形です。

## 役割
プレイヤーに見えてよいものだけを置きます。

入れるもの:
- HTML / CSS / JS
- プレイヤー向け導入文
- プレイヤー向け画像
- 回答欄

入れないもの:
- 正解条件
- GM判定プロンプト
- エンディング分岐
- 真相
- 未公開エピソード素材

## GitHub Pages 設定
GitHub の `Settings > Pages` で、公開元を `docs/` に設定してください。

## エピソード追加
`docs/episodes/ep02/` を作り、`episode.json` と画像を配置します。
`docs/episodes/index.json` にエピソードを追加します。
